Introduction to JOpt-Tour-Optimizer Demo Application

Goal of this application

This demo-application has the goal to familiarize you with JOpt-Tour-Optimizer. It shows a small part of the functional range. However, please visit our full Introduction on the JOpt optimization engine if you want to learn more about JOpt. Don't worry, for using this application it is not required.

  • Full introduction

    • How to use this application

    You can press Start Optimization right away...

    Implementation

    The application itself is implemented as an dockerized Angular Application. In the back it mainly uses Java, Spring Boot, and Swagger to perform the actual optimization. The front-end application is available as an open source project on GitHub .
